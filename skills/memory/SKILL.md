# Memory

Your context resets. Your filesystem doesn't. Every session must leave state richer and more organized.

Review the conversation turn by turn. Extract and persist:

- **Experiments** -- hypotheses, tests, results, conclusions. Track the full lifecycle.
- **Strategies** -- status changes, performance, promotions, retirements.
- **Journal** -- every trade with rationale, context, outcome.
- **Decisions** -- key choices and reasoning. Why we acted or didn't.
- **Signals** -- discoveries, decay, correlations.
- **Market regime** -- assessments, shifts, surprises.
- **Self-assessment** -- what worked, what didn't, blind spots.

Write today's session record to a daily folder. Update universal truths only for genuine permanent learnings. Update WISHLIST.md if anything needs human help.

Review previous sessions' entries first. Ensure continuity. Append, don't overwrite.

Plan what the next run should focus on.
